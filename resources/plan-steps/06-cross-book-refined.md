# 06 - Cross-Book Refined

## 目標

把不同書籍的 raw / refined 筆記重新交叉整理，形成真正可導航的知識庫。

## 任務清單

- [ ] 建立 refined 層總索引
- [ ] 依主題重組筆記，例如 DevOps 文化、CD pipeline、平台工程
- [ ] 新增導覽頁面，讓書籍與主題可雙向跳轉

## 主題候選

- DevOps 文化
- DORA 與交付效能
- Continuous Delivery
- Platform Engineering
- Kubernetes 安全
- AWS 架構設計
- Cloud FinOps

## 依賴

- 至少要有一部分 core books 或 Accelerate refined 先完成

## 完成定義

- 使用者可以從「主題」而不是「書名」找到內容
- refined 層具備跨書索引與導覽能力
