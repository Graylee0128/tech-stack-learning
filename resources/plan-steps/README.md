# Plan Steps

> 這個目錄放的是「可單獨派工」的 step files。

## 使用規則

- `plan.md` 管方向、優先順序、進度。
- `plan-steps/*.md` 管執行細節。
- 一次只把一個 step file 丟進 agent context，必要時再加 `00-transcription-sop.md`。

## 派工建議

| 檔案 | 最適合的工作 |
|------|--------------|
| `00-transcription-sop.md` | 共用轉錄規範 |
| `01-foundation.md` | 補工具、補模板、補 checklist |
| `02-accelerate-raw.md` | 逐章轉錄，可分 lane |
| `03-accelerate-refined.md` | 主題統整、面試整理 |
| `04-core-books.md` | 主線技術書排程 |
| `05-supplementary-books.md` | 次優先書籍排程 |
| `06-cross-book-refined.md` | 跨書交叉索引 |
| `07-phoenix-project.md` | 故事型 DevOps 書的輕量整理 |
