# 冷凍空調工程技師 學習與備考

> 目標：取得**冷凍空調工程技師**證照（考試院專技高考）

---

## 證照概覽

| 項目 | 說明 |
|------|------|
| **主管機關** | 考試院考選部 |
| **考試類型** | 專門職業及技術人員高等考試（專技高考） |
| **及格標準** | 平均 60 分；若及格人數未達全程到考 16%，取前 16% |
| **近年錄取分數** | 實際約 50~60 分可上榜 |
| **報名時間** | 每年約 8 月初，開放約 10 天 |
| **法規依據** | 專門職業及技術人員高等考試技師考試規則 |

---

## 應考資格（三擇一）

### 途徑 1：本科系畢業
公立或立案私立專科以上學校**冷凍空調工程科、系組所**畢業，領有畢業證書。

### 途徑 2：非本科 + 修學分（最常見）
專科以上學校畢業，修習相關學科：
- **總計**：≥ 5 科、≥ 15 學分（每科至多採計 3 學分）
- **必含**：下列 5 科中至少 **3 科**
  1. 冷凍空調（原理）
  2. 熱力學
  3. 流體力學
  4. 冷凍工程與設計
  5. 空調工程與設計

**開設學分班的學校：**
- 北部：北科大推廣教育中心、東南科大
- 南部：長榮大學

### 途徑 3：高等檢定
高等檢定考試相當類科及格。

> ⚠️ 以上資格僅供參考，最終須經考選部應考資格審查通過為準。

---

## 考試科目

共 **6 科**，全部為**申論題**，每科考試時間 2 小時。

| # | 科目 | 核心考點 |
|---|------|---------|
| 1 | **熱力學與熱傳學** | 熱力循環（朗肯、卡諾）、熱傳導/對流/輻射、封閉系統、稳流系統、理想氣體 |
| 2 | **流體力學與流體機械** | 伯努利方程式、文氏管、泵浦、風機、管路設計 |
| 3 | **冷凍工程與設計** | 莫利爾線圖（P-H 圖）、蒸氣壓縮循環、多段壓縮、冷媒特性、系統設計 |
| 4 | **空調工程與設計** | 空氣線圖（濕空氣線圖）、負荷計算、管路設計、冷卻水塔、AHU 設計 |
| 5 | **電工學（含電機機械）** | 電路分析、三相電機、馬達控制、電機原理 |
| 6 | **冷凍空調自動控制** | PID 控制、感測器、膨脹閥控制、BAS 系統 |

## 考試範圍與教材對照

### 本學習庫教材
**Stoecker & Jones《Refrigeration and Air Conditioning》2nd Ed.**（440 頁）

| 考試科目 | 對應章節 |
|---------|---------|
| 熱力學與熱傳學 | Ch2（熱力原理）、Ch10（蒸氣壓縮循環） |
| 冷凍工程與設計 | Ch10~14（壓縮機、冷凝器、蒸發器、膨脹裝置）、Ch15~16（冷媒、多壓力系統） |
| 空調工程與設計 | Ch3（濕空氣）、Ch4~5（負荷計算、空調系統）、Ch6~8（風管、水管、冷卻盤管） |
| 流體力學 | Ch7（泵浦與配管） |
| 自動控制 | Ch9（空調控制） |
| 進階主題 | Ch17（吸收式冷凍）、Ch18（熱泵）、Ch19（冷卻塔）、Ch20（太陽能）、Ch21（音響噪音） |

## 備考策略

1. **打理論基礎**：Stoecker & Jones 教材系統學習（本庫轉換計畫）
2. **刷歷屆試題**：100~115 年考古題（見 `exam-papers/` 目錄與執行計畫）
3. **弱點強化**：電工學通常非本科最弱科，需額外補強
4. **申論練習**：考試全為申論，需練習書寫格式與推導步驟
