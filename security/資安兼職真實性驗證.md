# 「學資安能兼職」— 真實性驗證

> 調查日期：2026-03-16
> 結論：**半真半假**，取決於你選哪種兼職模式

---

## 模式一：Bug Bounty（賞金獵人）— 高風險、高競爭

| 數據 | 說明 |
|------|------|
| HackerOne 註冊獵人 | 600,000 人 |
| 終身收入超過 $100K 的人 | 僅 **146 人**（0.025%） |
| 超過 $500K 的人 | 13 人 |
| 超過 $1M 的人 | 6 人 |

- 收入極度不穩定：某個月賺幾千美元，下個月幾乎零收入
- 報告可能被標為 duplicate（重複提交），白做工
- 幾百甚至上千人同時在同一個程式上找漏洞
- Google 2025 年付了 $1710 萬賞金，但只給了 747 人 — 頭部效應明顯

**結論：作為兼職收入來源極不可靠，除非你已是頂尖高手**

---

## 模式二：自由接案 / 顧問 — 這才是真正能兼職的路

| 類型 | 時薪/專案費 | 門檻 |
|------|-----------|------|
| 滲透測試（Pentest） | $60-150+/hr | OSCP + 實戰經驗 |
| 安全稽核/合規顧問 | $5,000-20,000/專案 | 熟悉 ISO 27001, SOC2 等框架 |
| 雲端安全評估 | $80-150/hr | AWS/Azure 安全證照 + 實務 |
| 權限審查/SIEM 調校 | 按專案計費 | 藍隊實戰經驗 |

- 企業有定期合規需求（每 6-12 個月一次稽核）
- 中小企業請不起全職資安，但願意外包
- 時間彈性，按專案走
- 平台：Upwork, Toptal, Fiverr

---

## 模式三：教學 / 內容創作 — 長期被動收入

| 類型 | 收入方式 | 適合誰 |
|------|---------|--------|
| 線上課程（Udemy, Teachable） | 被動收入，單課 $500-5000+/月 | 有教學能力 |
| 技術寫作/部落格 | 廣告、贊助、聯盟行銷 | 願意長期經營 |
| YouTube / 教學影片 | 廣告收入 + 課程導流 | 不怕露臉 |
| CTF Write-up | 建立個人品牌 → 接案管道 | 有 CTF 經驗 |

---

## 模式四：兼差/Part-time 職位

- SOC 輪班監控（夜班/週末班）
- 資安教育訓練講師（1-2 小時/週）
- 遠端 code review / security review

---

## 總結

| 模式 | 可行性 | 收入穩定度 |
|------|--------|-----------|
| Bug Bounty | ⚠️ 低（除非頂尖） | 極不穩定 |
| **安全顧問/接案** | ✅ **最實際** | 中等 |
| **教學/內容** | ✅ 可行 | 慢慢累積 |
| Part-time 職位 | ✅ 可行 | 穩定 |

**「學資安能兼職」這句話本身沒錯，但 Bug Bounty 不是大多數人的答案，安全顧問和教學才是。**
