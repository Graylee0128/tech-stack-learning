# AWS 網路解題紀錄

集中記錄做題時遇到的題目、錯因、正確判斷依據與易混服務，方便之後做錯題複盤。

## 使用方式

- 每次做題後，新增 1 份 `questions/YYYY-MM-DD.md`，或補到當天同一份檔案。
- 每題至少記下：`主題分類`、`題幹關鍵字`、`正解`、`錯因`、`正確判斷依據`、`錯誤選項分析`、`陷阱`。
- 若同類錯誤累積 3 次，就回補對應 `ans-notes` 或 `notes`。
