# AWS 網路學習紀錄

按日期記錄 AWS Advanced Networking Specialty 的學習內容，方便依主題與日期回顧。

## 使用方式

- 每次學習後新增 1 份 `logs/YYYY-MM-DD.md`
- 記錄當天讀了哪些章節、學到哪些關鍵判斷、做了哪些題目
- 若有錯題，直接記下錯因：服務不熟 / 路由判斷錯 / 題幹關鍵字漏看
- 若有 lab 或架構演練，補上 1 個選型理由與 1 個常見陷阱
