# Zeroclaw 詳細分析

## 📌 專案概述

**Zeroclaw** 是一個開源開發工具專案，專為快速開始和靈活配置設計。強調 **易用入門體驗** 和 **多提供商支援**，擁有業界最高的社群關注度（21.8k+ 星標）。

**GitHub**: zeroclaw-labs/zeroclaw
**星標**: 21.8k+ ⭐
**Forks**: 2.7k+
**授權**: 開源（具體待驗證）

## 🎯 核心功能

### 主要命令
```bash
zeroclaw onboard      # 配置與初始化（支援多種設定路徑）
zeroclaw chat         # 互動式對話
zeroclaw status       # 狀態檢查
zeroclaw doctor       # 環境診斷與驗證
```

### 配置模式
1. **快速模式** - 有 API 密鑰，想要最快速的設定
2. **互動模式** - 引導式提示符流程，逐步配置
3. **OAuth 模式** - OpenAI Codex 訂閱認證方式

### 跨平台支援
✅ **多系統支援**：
- macOS
- Linux
- Android（透過 Termux/ADB）
- Windows（待驗證）

### AI 模型支援
- OpenAI Codex / GPT 系列
- Anthropic Claude（推測）
- 多提供商切換

## 💾 部署與維護

### 安裝難度 - ⭐⭐ 中等
```bash
# 典型安裝流程
1. 安裝 Zeroclaw CLI
2. zeroclaw onboard （快速或互動配置）
3. zeroclaw chat （開始互動）
```

### 配置複雜度
- **快速路徑**: 中等（需要 API 密鑰）
- **互動路徑**: 低（引導式提示）
- **OAuth 路徑**: 中（需要理解 OpenAI Codex）
