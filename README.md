# Tech Stack Learning 📚

我的技術棧學習記錄庫。系統性記錄學習過程中的筆記、最佳實踐和實戰項目。

## 📁 目錄結構

### 工具筆記
按工具/技術名組織，每個目錄包含：
- 基礎概念
- 常用命令
- 配置示例
- Q&A 和解決方案

目前包括：
- `docker/` - 容器化
- `git/` - 版本控制
- `terraform/` - 基礎設施即代碼
- `aws/` - AWS 相關
  - `network/` - AWS 網路服務
  - `security/` - AWS 安全服務

### 其他重要目錄
- **[learning-path/](./learning-path/)** - 學習路線圖和規劃
- **[projects/](./projects/)** - 實戰項目記錄
- **[cheatsheet/](./cheatsheet/)** - 速查表和常用命令

## 🎯 目的

建立一個個人的、系統化的技術棧知識庫，幫助：
- 📖 記錄學習過程
- 🔍 快速查詢已學知識
- 📈 追蹤學習進度
- 🤝 便於未來分享和協作

## 📝 如何使用

1. **查詢特定工具** - 進入對應目錄，查看 Q&A 或筆記
2. **了解學習路線** - 查看 `learning-path/` 了解推薦的學習順序
3. **實踐項目** - 參考 `projects/` 下的項目案例
4. **快速查詢** - 使用 `cheatsheet/` 找常用命令
